The portal logo (public/assets/portal-logo.svg) is a custom SVG design created
for this project. It does not use any official Indian national or state police
emblems, seals, or marks.

## Logo Design

The portal logo uses:
- A shield shape with checkmark (public domain iconography)
- India tricolor accent: saffron (#FF9933), white (#FFFFFF), green (#138808)
- No Ashoka Chakra, State Emblem of India, or any state police marks

This is compliant with:
- The State Emblem of India (Prohibition of Improper Use) Act, 2005
- The Emblems and Names (Prevention of Improper Use) Act, 1950
- MHA Emblem Rules 2007

## Adding External Images

When adding stock photos later:
1. Download from Pexels, Unsplash, or Wikimedia Commons
2. Place files in public/assets/
3. Add a row to the images table above with:
   - Filename
   - Source URL
   - Author/Photographer
   - License type (e.g., Pexels Free License, Unsplash License, CC BY-SA 4.0)
   - Any attribution requirements
